使用 React 定义 UI 结构：使用 JSX 语法编写一个组件，结构由 React 核心库创建并管理。

Babel 实时编译：由于浏览器无法直接识别 JSX，babel-standalone 会拦截带有 type="text/jsx" 属性的 <script> 标签，并实时地将 JSX 代码编译成标准的 JavaScript。

ReactDOM 渲染到页面：编译完成后，ReactDOM.render(...) 方法被调用，将 React 组件生成最终的 HTML 元素并真实地“画”在网页的指定容器内。
